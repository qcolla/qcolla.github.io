Dossier documentaire

Pour constituer ce dossier documentaire sur la recherche documentaire sur smartphone, les bases de données suivantes ont été interrogées : Google,  Doctes (base de données duRESOdoc -http://superdoc.reso.ucl.ac.be/superdoc/Doctes/search.htm)  et Cairn info.

Pour trouver la documentation adéquate les mots clefs suivant ont été utilisés : « Smartphone », « Bibliothèque »,  « Danger »,  « Recherche Documentaire », « La recherche documentaire sur smartphone »

Ce dossier sera classé en plusieurs grands thèmes.Le smartphone ( quelques définitions), la recherche documentaire sur smartphone en bibliothèque, la recherche documentaire sur smartphone, des ouvrages généraux sur le numérique chez les jeunes, l'éducation aux médias, la formation à la recherche documentaire et la formation à l'utilisation du smartphone
